# 自动将 markdown 文件转换为 Posts（博客）

命名格式为 `YYYY-MM-DD-*.md`。 例如:

```shell
2020-01-28-My-First-Post.md
2012-09-12-how-to-write-a-blog.md
```
支持中文。

- 当心错误的命名!  很容易将 `YYYY-MM-DD-`的最后一个破折号落下。 并且，破折号后面的一个字符只能是文字而不能是符号。
